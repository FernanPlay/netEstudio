# netEstudio

netEstudio es una pagina web enfocada en el estudio. La idea del proyecto es crear una plataforma simple, ordenada y facil de usar para que los estudiantes puedan acceder a herramientas y funciones que ayuden en su aprendizaje.

## Objetivo

Crear una pagina para estudio que pueda crecer poco a poco con nuevas funciones, manteniendo un diseno claro para computadora y celular.

## Funciones actuales

- Pantalla de inicio de sesion responsive.
- Pagina de registro para crear usuarios.
- Dashboard simple para usuarios con sesion iniciada.
- Logo personalizado con tema FP.
- Icono de la pagina usando el mismo logo.
- Boton para mostrar u ocultar la contrasena.
- Notificacion cuando el usuario o la contrasena son incorrectos.
- Guardado de usuarios en `assets/bd/datos.db` usando SQLite.
- Cierre de sesion desde el dashboard.
- Version visible de la pagina: `0.0.1`.

## Paginas

- `login.html`: pagina de inicio de sesion.
- `register.html`: pagina de registro.
- `dashboard.html`: panel simple despues de iniciar sesion.

## Funciones planeadas

- Panel principal para organizar contenido de estudio.
- Secciones para tareas, notas o recursos.
- Mejoras visuales y nuevas herramientas utiles para estudiantes.
- Evolucion por versiones conforme se agreguen funciones.

## Tecnologias usadas

- HTML
- CSS
- JavaScript
- PHP
- SQLite

## Uso local

Para que el registro y el login funcionen con `datos.db`, abre el proyecto desde WAMP o un servidor PHP local. Si abres los archivos directamente con doble clic, el diseno se vera, pero no se podra guardar ni leer la base de datos.
